* header
* 請使用header標籤，不要使用 div class="header"
* 網頁頭註解請放在header標籤上方，並切齊body標籤
* 全世界都header都應該相同，除了辨別該頁的特效
* main
* 最外層請使用main標籤，不要使用 div class="container"
* 網頁內容註解請放在main標籤上方，並切齊body標籤，且與header間格2個空白行

* ===== 文件路徑放置規則 ===== *
* 請不準、不要、不可以使用中文命名資料夾及檔案
* 所有路徑請使用相對路徑抓取
* html檔請不要放置在其他子資料夾內
* js檔請放置在 js資料夾內
* css檔請放置在 css資料夾內
* 圖片檔請放置在 images資料夾內
* 圖片檔命名格式： 使用頁面_使用區塊(class)_icon/bg/X.副檔名
- 使用頁面: 就是哪個html檔 - 使用區塊: 可以以使用那張圖的class大區塊命名，需有意義可以辨認是哪塊 - icon/bg/x: 可以不寫或是依圖示或背景命名 - e.g. 首頁開場動畫背景 -> homepage_animation_bg.jpg - e.g. 會員大頭貼 -> member_sticker.jpg - e.g. 商城商品 -> shop_tarot_thefool_detail.jpg * 影片、影音檔請放置在 media資料夾內

* ===== 標籤、id、class命名規則 ===== *
* 請不準、不要、不可以使用中文命名class、id
* 所有的複合字請使用 _ 串接 e.g. class="many_words"
* id、class="請使用雙引號包"
* 容器與內容分開，除了以下容器以外，容器後面加上 _box
- section標籤
- container
- row
* 每個大區塊請以section作為標籤
* section寬度固定100%
* 每個section內最少要有一個容器，可用來控制寬度(padding:8.333333%)
* 卡片式排版，容器請以 item_box 命名，卡片以 item 命名
* 圖片容器請固定使用 img_box
* 文字容器請固定使用 txt_box
* 清單請使用 ul/ol li 標籤

* 圖片要加破圖註解(alt)
* 命名請避免使用數字編號

* 每個section間請留1個空白行 [4]
